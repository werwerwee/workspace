Identifier - 식별자

변수 - variable

컴퓨터 기억장치인 메모리상에 사람이 식별할 수 있는 이름을 부여하여 데이터를 저장하는 공간

식별자 - 작성 규칙

어떤 데이터나 함수를 식별하는데 사용되는 이름

식별자 생성 규칙 - 문법 오류 - 프로그램 종료
  • javascript 예약어 - 키워드는 사용불가
  • 특수문자는 _ (언더바, 언더스코어) 또는 $ 기호만 사용가능
  • 숫자를 사용가능하나 숫자로 시작할 수는 없음
  • 식별자 사이에 공백은 입력할 수 없음
식별자 작성 규칙

문법 오류는 아니나 프로그래밍 언어에서 변수나 함수등의 개발자 사이에 소통을 위해 지켜지는 규칙

식별자 이름만 보더라도 변수 함수 객체인지 또는 어떤걸 의미하는 등의 정보를 이해할 수 있도록 명명하는 표기법
